Key Commands

.. csv-table::
   :header: "Key Command", "Description"

   "-++ /-++", "Open launcher settings"
   "-++ / -++ / +++-", "Factory reset"

*Key combo only works for Robox version 128 and above, the "-" and "+" stand for the volume key, and "/" is a short pause of around 0.5 seconds.*
