*******************
Roles & Permissions
*******************

Once a robot is registered, it will have 1 owner. It can also have multiple Contacts and/or Members. Whether or not the robot has Contacts and/or Members depends on the owner.


Owner
=====
An robot's owner is the user that activates/registers the robot.

The owner's mobile phone's contact list is used to automatically assign robot Contacts.

Owners can make any one of their Contacts a Member of their robot(s) by assigning them one of various permissions.


Contacts
========
Contacts from the owner's mobile phone's contact list are automatically made temi Contacts if the Contact is already registered with temi's Mobile App.

Like a telephone, it is possible for a Contact to start a call (or a telepresence session) with a robot, but someone will need to accept the call.


Member
======
Owners can make any one of his/her Contacts a Member of any one of his/her robots. Each member has a set permissions assignable by the robot's owner.

- Call Through / Hop-In 
- Control 
- Edit Map 
- Edit Settings 

Call Through / Hop-In 
---------------------
A member can start a telepresence session with a robot without the need for call acceptance.

Control 
-------
A member can control a robot using the Control view in temi Center.

Edit Map 
--------
A member can edit the map of a robot using temi Center's Map Editor.

Edit Settings 
-------------
A member can remotely edit the robot's settings using temi Center.
